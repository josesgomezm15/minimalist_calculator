🧮 Calculadora Minimalista

Aplicación web de calculadora desarrollada con HTML, CSS y JavaScript, con un diseño moderno tipo minimalista (neumorphism).

📌 Descripción

Este proyecto consiste en una calculadora funcional con operaciones básicas, diseñada con una interfaz limpia, moderna y amigable para el usuario.

Incluye lógica propia en JavaScript sin el uso de funciones inseguras como eval().

🚀 Tecnologías utilizadas
HTML5
CSS3 (Grid + Neumorphism)
JavaScript (Vanilla)
🎨 Características
🧮 Operaciones básicas: suma, resta, multiplicación y división
🎯 Interfaz minimalista y moderna
💡 Diseño con sombras suaves (neumorphism)
❌ Sin uso de eval() (más seguro)
🔁 Botón de borrar (⌫)
🧹 Botón de limpiar (C)
📱 Responsive (adaptable a diferentes pantallas)
